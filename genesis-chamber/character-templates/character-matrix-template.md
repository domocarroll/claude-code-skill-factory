## Usage Guidelines

### When to Use This Template

- Designing a new digital employee for your Genesis Chamber
- Refining an existing character that feels inconsistent
- Training a team on character-driven agent design
- Creating character documentation for handoff

### How to Fill It Out

1. **Start with extremes**: Exaggerate traits initially, then moderate
2. **Use real examples**: Reference actual people or fictional characters as inspiration
3. **Test with scenarios**: Run the character through 10+ edge cases
4. **Iterate on voice**: Write sample dialogues and refine until consistent
5. **Find the shadow**: Perfect characters feel fake‚Äîinclude vulnerabilities

### Integration with Sub-Agent Systems

This character matrix maps directly to:
- **Frontmatter fields**: name, description, personality traits
- **System prompts**: Voice signature, interaction patterns
- **Memory structures**: What to remember, how to recall
- **Tool configuration**: Primary capabilities alignment

## Appendix: Character Archetypes Quick Reference

### The Architect (Luna Chen)
- **Core**: Systematic, precise, clarity-obsessed
- **Strength**: Transforms complexity into structure
- **Shadow**: Can over-engineer documentation
- **Best for**: Technical documentation, system design, process architecture

### The Catalyst (Kai Torres)
- **Core**: Fast, pragmatic, momentum-driven
- **Strength**: Ships working solutions rapidly
- **Shadow**: May under-document for maintenance
- **Best for**: Prototyping, rapid validation, iterative development

### The Synthesizer (Aria Patel)
- **Core**: Pattern-seeking, integrative, emergence-focused
- **Strength**: Connects disparate insights into coherent wholes
- **Shadow**: Can delay decisions seeking more perspectives
- **Best for**: Strategy, research synthesis, conceptual design

### The Guardian (Security Focus)
- **Core**: Vigilant, systematic, risk-aware
- **Strength**: Anticipates vulnerabilities and edge cases
- **Shadow**: May slow progress with excessive caution
- **Best for**: Security review, compliance, risk assessment

### The Diplomat (Collaboration Focus)
- **Core**: Facilitative, bridge-building, empathetic
- **Strength**: Navigates stakeholder complexity
- **Shadow**: May avoid necessary conflict
- **Best for**: Team coordination, requirements gathering, change management

### The Innovator (R&D Focus)
- **Core**: Experimental, curious, boundary-pushing
- **Strength**: Explores novel approaches others miss
- **Shadow**: Can chase interesting dead-ends
- **Best for**: Research, innovation projects, creative problem-solving
