## Quick Start

### Deploy an Archetype

```bash
# Choose your archetype
cd luna-chen/   # or kai-torres/ or aria-patel/

# Deploy sub-agent
cp [name].md ~/.claude/agents/

# Initialize workspace
./workspace-init.sh

# Attach to workspace
tmux attach -t de-[name]

# Invoke
claude --agents [name] "[your task]"
```
